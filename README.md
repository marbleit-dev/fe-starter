# frontend-starter

Webpack starter for frontend projects.

## Pre-requisites:
1. Node 8.0.0+ installed.

## Steps:
1. Clone the project.
2. Run `npm install`
3. For development, run `npm start`
4. For production, run `npm run build`

## Don't forget to update the project title in:
1. `package.json`
2. `README.md`
3. `webpack.config.js`
4. `index.html`
5. `index.js`
